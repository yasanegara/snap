📋 Cara pasang pakai Cloudflare (gratis)

  1. Bikin akun gratis di cloudflare.com, klik "Add a Site", masukin domain kamu.
  2. Cloudflare bakal kasih 2 nameserver (contoh: ana.ns.cloudflare.com). Copy itu.
  3. Buka tempat kamu beli domain (Niagahoster, Rumahweb, GoDaddy, dll), cari menu "Nameserver" / "DNS", ganti nameserver-nya pakai punya Cloudflare tadi.
  4. Balik ke Cloudflare, buka menu DNS, tambah record baru:
    • Type: CNAME
    • Name: @ (atau www kalau mau pakai www)
    • Target: (domain Railway kamu, cek di dashboard Railway)
    • Proxy status: pastikan awan ORANYE nyala (Proxied) — ini kuncinya, biar SSL otomatis jalan tanpa perlu daftar manual ke Railway.
  5. Buka menu SSL/TLS di Cloudflare, pilih mode "Full".
  6. Tunggu 5-30 menit buat propagasi. Domain siap dipakai, SSL (https) otomatis aktif gratis dari Cloudflare.

💡 Kalau domainnya udah lama dipakai Cloudflare buat hal lain, cukup tambah record CNAME-nya aja (lewatin langkah ganti nameserver).
